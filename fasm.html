Язык Ассемблера — это низкоуровневый язык программирования различный для разных архитектур. Язык ассемблера переводится в машинный код с помощью компиляторов ассемблера.

Все релизы

Секции

Программы на ассемблере могут быть разделены на три секции: bss, data, text

Секция data требуется для объявления констант или данных.

section.data

Секция bss для объявления переменных.

section.bss

Секция text содержит код программы.

section.text

Комментарии

Комментарии начинаются с ";", а все после ; до конца строки не будет исполнено.

section.text ; Секция text ;section.bss - вот так объявляется секция bss

Операторы

Операторы(или стейтменты) бывают трех видов:

  • Инструкции.
    • Сам код. Сообщает процессору что надо делать.
  • Директивы.
    • Сообщения компилятору об аспектах компиляции.
  • Макросы.
    • Простой механизм вставки кода.
